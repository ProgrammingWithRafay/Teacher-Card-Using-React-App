# Teacher App

A React-based web application showcasing teacher profiles with their subject expertise, experience, and bio information. This app displays teacher cards with profile pictures, contact details, and educational background.

## Features

-  Display teacher profiles with detailed information
-  Show teacher name, subject, experience, and bio
-  Contact information for each teacher
-  Profile pictures for each teacher
-  Clean and responsive UI with CSS styling

## Tech Stack

- **Frontend Framework:** React
- **Styling:** CSS
- **Language:** JavaScript (ES6+)
